# Sales-Dashboard-Excel
## 🌐 Project Overview
This project presents an interactive sales dashboard created in Microsoft Excel to analyze sales performance across different categories, channels, states and months.

## 📈 Dashboard Features 
- Total Sales and Total Summery.
- Sales analysis by Months.
- Sales by Channel (Amazon, Flipkart etc.).
- Top 5 States by Sales Performance.
- Product Category-wise Sales.
- Interactive filters using slicer.

## 📊 Insights
- Women Contributes the majority shares of total sales.
- Amazon is the leading Sales Channel.
- March shows the highest monthly sales.
- Sales show slight decline in the last quarter months.
- Maharashtra and Karnataka generate the highest revenue.
